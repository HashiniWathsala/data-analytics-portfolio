# Cancer Outcomes and Poverty — Regression Analysis

## Overview
This project investigates whether economically deprived regions experience higher cancer mortality and fewer clinical trials. Using county-level data from the United States, multiple regression models were developed to examine the relationship between poverty and two key outcomes: cancer death rates and clinical trial availability.

## Objectives
- Analyze the relationship between poverty and cancer mortality
- Examine whether poorer regions receive fewer clinical trials
- Build and evaluate multiple regression models
- Interpret statistical significance and model fit

## Dataset
The dataset contains socioeconomic and health indicators for U.S. counties, including:

- Cancer death rate
- Clinical trials per capita
- Poverty percentage
- Median income
- Education levels
- Population estimates
- Cancer incidence rates

## Methods
- Exploratory Data Analysis (EDA)
- Scatterplot visualization
- Multiple linear regression
- Backward stepwise selection (AIC)
- Residual diagnostics
- Model evaluation using Adjusted R² and F-tests

## Key Findings
- Higher poverty is associated with higher cancer death rates
- Education is a strong protective factor against mortality
- Poorer counties host fewer clinical trials
- Trial distribution is highly concentrated in major research hubs
- The mortality model explained substantial variation, while the trial model explained little

## Tools Used
- R
- tidyverse (dplyr, ggplot2)
- Base R statistical modeling
- Quarto
